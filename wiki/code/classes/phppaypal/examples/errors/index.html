If there was an error with the transaction, the variable _error will be set to true.

$paypal->_error = true;

The following error fields will be filled (according to the type of error) when _error returns true.

NOTE: These fields also contain the same values as the PayPal Response, except for _error_corrective_action and _error_display_message. These are custom values according to the error code (these are not yet working as of v0.5).

public $_error = true;
public $_error_ack;
public $_error_type;
public $_error_date;
public $_error_code;
public $_error_short_message;
public $_error_long_message;
public $_error_corrective_action; // Not used as of v0.5
public $_error_display_message; // Not used as of v0.5
public $_error_severity_code;
public $_error_version;
public $_error_build;

phpPayPal

(PHP 5) - This has not been tested with PHP 4.

You can download this class from the Github project downloads page.

To report an issue, please go to the Github project issues page.

Description

phpPayPal ( )

phpPayPal contains the variables and methods needed to make interactions with PayPal.

The methods currently implemented are:

  • DoDirectPayment
  • SetExpressCheckout
  • GetExpressCheckoutDetails
  • DoExpressCheckoutPayment
  • GetTransactionDetails
  • RefundTransaction

Scheduled for later are:

  • DoAuthorization
  • DoCapture
  • DoReauthorization
  • DoVoid
  • TransactionSearch
  • Mass Payment Methods
  • Recurring Payment Methods
  • DoReferenceTransaction

Class Parameters

NULL - This class does not require any parameters to create an instance. Username, password and certificate information are currently embedded in the class itself.

Return Values

NULL - This class does not return a value upon creating an instance.

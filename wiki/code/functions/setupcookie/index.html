setupCookie

(PHP 4, PHP 5)

setupCookie - Setup the local cookie for the site

Description

NULL setupCookie ( array $Settings )

setupCookie will check and either set a none existing cookie or confirm there is one already set.

Parameters

Settings

  • An array of site/application settings setup at the beginning of a script execution. See Settings Array

Return Values

NULL - This function does not return a value.

Examples

Example of how to execute this function. See [[code:functions:settings_array|Settings Array]] to understand what is being passed.
